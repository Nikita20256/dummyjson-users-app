# DummyJSON Users Management App

Приложение для просмотра, фильтрации и сортировки списка пользователей из API DummyJSON.

## 🎯 Возможности

- 📋 Просмотр списка пользователей в интерактивной таблице
- 🔍 Фильтрация по фамилии, имени, возрасту, полу и стране
- 🔄 Сортировка по всем столбцам (возрастание, убывание, сброс)
- 📄 Пагинация с 8 пользователями на странице
- 👤 Модальные окна с полной информацией о пользователе
- 🎨 Современный красивый дизайн
- 📱 Полностью адаптивный интерфейс

## 🛠️ Технологический стек

- **React** 18.3.1 - библиотека для создания UI
- **Vite** 6.0.7 - быстрая сборка и разработка
- **JavaScript ES6+** - современный JavaScript
- **CSS3** - стили с градиентами и анимациями

## 📁 Структура проекта

## 🚀 Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Приложение откроется на `http://localhost:5173/`

### Сборка для production
```bash
npm run build
```

### Предпросмотр собранного приложения
```bash
npm run preview
```

## 📊 Функциональность

### Таблица пользователей
- Отображает данные: фамилия, имя, возраст, пол, телефон, email, страна, город
- Красивые стрелки сортировки с анимацией
- Клик по строке открывает полный профиль

### Фильтры
- Фильтрация по фамилии (частичное совпадение)
- Фильтрация по имени (частичное совпадение)
- Фильтрация по точному возрасту
- Фильтрация по полу
- Фильтрация по стране (частичное совпадение)
- Кнопка "Очистить фильтры"

### Профиль пользователя
- Фото пользователя
- Полная информация о пользователе
- Красивые карточки с деталями
- Удобное закрытие (крест, фон или Escape)

## 🌐 API

Приложение использует бесплатный API: [DummyJSON](https://dummyjson.com/)

Endpoint: `https://dummyjson.com/users`
